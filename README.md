# Introduction

All data label terms documents stewarded by Movement for an Open Web are located
in this repository.

For more information concerning data labels see the
[data-labels](https://github.com/jwrosewell/data-labels) repository.

## Naming

Each folder is a lower case acronym for the document title. For example Search
Only Terms Web is `socw`.

Active files are in plain text format and named with numbers indicating their
relative version.

Numbered documents are immutable and must not change once in use. Therefore if a
change is required a new file will be created with the number incremented by
one. 1 becomes 2, 2 becomes 3, and so on. This is essential to ensure that those
using data covered by the terms documents can control any changes to their use,
and those that rely on the enforcement of the terms documents can point to a
version that is easily identifiable by non-technical people.

Names should use as few characters as possible to improve technical efficiency
where terms might be attached to very large numbers of data exchanges. For
example; as headers in HTTP.

## Referencing

The main branch of the repository is published at the domain `m4ow.uk`.
Therefore to access the first version of the Search Only Contract Web (`socw`)
the TDL or URL will be <https://m4ow.uk/socw/1.txt>.

Those referencing terms documents in this repository must refer to a number
document.

## Changes

To be notified of changes watch this repository. This is the best method of
learning when documents have been superseded with a new version.

Differences between documents will be explained in the commit history for each
document.

## Questions

Any questions about documents in this repository should be raised via GitHub
issues.
